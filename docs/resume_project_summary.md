# Resume Project Summary

## 项目简述

**政务热线工单 RAG 数据集标准化工程**  
围绕政务热线物业问题分类与标签推荐场景，完成原始标注 JSONL 的字段盘点、schema 重构、标准化转换、train/validation/test 划分、manifest 设计、旧数据归档、新旧映射、质量验收和文档生成。设计仅对问题描述字段进行 embedding、标签与理由作为 payload 保留的 RAG 数据策略，形成可用于 Qdrant 入库、相似工单召回和模型评估的数据底座。项目沉淀了 Notebook 自动化流程、标准 JSONL 模板、数据集说明文档和脱敏 GitHub 展示仓库。

## 关键词

- 数据集工程；
- RAG 数据治理；
- JSONL schema 设计；
- 数据质量验收；
- 向量检索 payload 设计；
- 敏感数据脱敏与公开项目包装；
- Notebook 自动化流程沉淀。
